HTML Table Styling

Table Tags

Tag Name Description
<table> Table A wrapper element for all HTML Tables. This creates a 2D table made of rows and columns
<thead> Table Head Defines the headings of table columns encapsulated in table rows
<tbody> Table Body An element that will contain all table data other than table heading and table footer content
<tr> Table Row An element used to to add rows to a table before adding table data
<td> Table data can be nested inside a table row element, to add a cell of data to a table
<tfoot> Table Foot Uses table rows to give footer content or to summarize content at the end of a table

Table Attributes

Tag Name Description
colspan Column Span An element that indicates how many columns that particular cell should span within the table, default of 1
rowspan Row Span An element that indicates how many rows that particular cell should span within the table, default of 1
